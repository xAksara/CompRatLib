Provides a proxy object to manipulate a slice of a larger matrix. 


\begin{DoxyTemplParams}{Template Parameters}

\hline
{\em container\+\_\+T} & The type of container used to store matrix elements (default\+: std\+::map\texorpdfstring{$<$}{<}std\+::pair$<$unsigned int, unsigned int$>$, element\+\_\+T\texorpdfstring{$>$}{>}).\\
\hline

The {\ttfamily \doxylink{classMatrix__proxy}{Matrix\+\_\+proxy}} class represents a submatrix (slice) of a larger matrix. It enables both read and write access to elements within the defined slice. Changes made to the proxy are reflected in the original matrix unless a local copy is made.
