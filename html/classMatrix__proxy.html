Detailed Description

template<typename element_T = double, typename container_T = std::map<std::pair<unsigned int, unsigned int>, element_T>>
class Matrix_proxy< element_T, container_T >

Provides a proxy object to manipulate a slice of a larger matrix.

Template Parameters
element_TThe type of elements stored in the matrix (default: double).
container_TThe type of container used to store matrix elements (default: std::map<std::pair<unsigned int, unsigned int>, element_T>).

The Matrix_proxy class represents a submatrix (slice) of a larger matrix. It enables both read and write access to elements within the defined slice. Changes made to the proxy are reflected in the original matrix unless a local copy is made.
