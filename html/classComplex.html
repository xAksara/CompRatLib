Constructs a complex number from a string representation.

Parameters
strA C-style string representing the complex number.
Exceptions
InvalidArgumentIf the string does not represent a valid complex number.

The input can be in the following formats:

  • real+imaginary*i or real-imaginary*i
  • real,imaginary (comma-separated format)
  • A single real number without an imaginary part.
